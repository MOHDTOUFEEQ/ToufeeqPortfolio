# 🚀 **Personal Portfolio**

A personal portfolio website showcasing my skills, experience, and projects as a **Full-Stack Developer**. Built with **React.js**, **Tailwind CSS**, **HTML**, **CSS**, and **GSAP** animations, this portfolio provides an interactive, responsive, and visually appealing experience.

## 📌 **Features**
- **Responsive UI**: A clean and professional design using **Tailwind CSS**, ensuring a smooth experience across all devices.
- **GSAP Animations**: Engaging animations and transitions for an interactive experience.
- **Project Showcases**: Displaying key personal and freelance projects with descriptions, technologies used, and links.
- **About Me & Skills**: Information about my background, skills, and expertise.
- **Contact & Social Links**: Easy way to get in touch with links to email and social media profiles.
